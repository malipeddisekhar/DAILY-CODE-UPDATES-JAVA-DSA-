## 🚀 How to Use This Repository

### **For Interview Preparation:**
1. **Study the Pattern**: Understand the algorithm pattern (Two Pointers, DP, Sliding Window, etc.)
2. **Solve on Your Own**: Attempt the problem without looking at the solution
3. **Review the Solution**: Compare your approach with the provided solution
4. **Analyze Complexity**: Study time and space complexity analysis
5. **Practice Variations**: Solve similar problems to strengthen the pattern

### **Clone and Run:**
```bash
# Clone the repository
git clone https://github.com/yourusername/CRT.git
cd CRT

# Navigate to specific day/problem
cd "DSA(STRIVE SHEET)/Day-1"

# Compile a Java file
javac Next_Permutation.java

# Run the solution
java Next_Permutation
```

### **Study Method:**
```bash
# For each problem:
1. Read the problem statement (comments in code)
2. Understand the approach used
3. Trace through with the given test cases
4. Understand Time Complexity: O(?) and Space Complexity: O(?)
5. Think of edge cases and optimizations
6. Implement variations or solve similar problems
```
